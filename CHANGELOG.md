# Version History - MAS + ARCH

## [1.2.0] - Production Ready Version (Current)
### Added
- **Description System:** Projects now display technical and narrative info in the selected language upon hover.
- **Fluid Typography:** Implemented `clamp` in the Hero section for universal adaptability.
- **100rem Alignment:** Unified global containers across Hero, Projects, Contact, and Footer.
- **CSS Keyframes:** Migrated Hero animations to pure CSS to prevent flickering during language switching.

### Optimized
- **Styles.css:** Centralized animations, minimalist scrollbar, and UI protection for the mobile menu.
- **Performance:** Integrated `will-change` on reveal elements to boost FPS on mobile devices.

## [1.1.0] - Internationalization & Dark Mode
- Implemented language switching logic (ES/EN) via `localStorage`.
- Dynamic color system for Dark Mode (`#1A1C1E`).
- `works` data object restructured to support multi-language strings.

## [1.0.0] - Base Structure
- Initial layout with Tailwind CSS.
- Core section configuration: Hero, Services, Projects, and Contact.
- Smooth scroll logic and hamburger menu implementation.


# Historial de Versiones - MAS + ARCH

## [1.2.0] - Versión Final de Producción (Actual)
### Añadido
- **Sistema de Descripciones:** Los proyectos ahora muestran información técnica y narrativa en el idioma seleccionado al hacer hover.
- **Fluid Typography:** Implementación de `clamp` en el Hero para adaptabilidad universal.
- **Alineación 100rem:** Unificación de contenedores globales en Hero, Projects, Contact y Footer.
- **CSS Keyframes:** Animaciones del Hero migradas a CSS puro para evitar parpadeos en el cambio de idioma.

### Optimizado
- **Archivo Styles.css:** Centralización de animaciones, scrollbar minimalista y protección de UI para menú móvil.
- **Rendimiento:** Uso de `will-change` en elementos de revelado para mejorar los FPS en móviles.

- Implementación de lógica de cambio de idioma (ES/EN) mediante `localStorage`.
- Sistema de colores dinámicos para modo oscuro (`#1A1C1E`).
- Estructura de objetos `works` separada por idiomas.

## [1.0.0] - Estructura Base
- Layout inicial con Tailwind CSS.
- Configuración de secciones: Hero, Services, Projects, Contact.
- Lógica de scroll suave y menú hamburguesa.
